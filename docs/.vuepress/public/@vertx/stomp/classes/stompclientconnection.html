Class StompClientConnection

Once a connection to the STOMP server has been made, client receives a StompClientConnection, that let send and receive STOMP frames.

receivedFrameHandler

  • receivedFrameHandler(handler: function | Handler<Frame>): StompClientConnection
    • Defined in index.d.ts:858

    Configures a received handler that get notified when a STOMP frame is received by the client. This handler can be used for logging, debugging or ad-hoc behavior. The frame can still be modified by the handler.

    Unlike {@link StompClient#receivedFrameHandler(Handler)}, the given handler won't receive the {@code CONNECTED} frame. If a received frame handler is set on the StompClient, it will be used by all clients connection, so calling this method is useless, except if you want to use a different handler.

    Parameters

    • handler: function | Handler<Frame>

      the handler

    Returns StompClientConnection
